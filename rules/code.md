# Pravidla pro kód

- Používej TypeScript místo JavaScriptu kde je to možné
- Vždy přidej typy — žádné `any`
- Komentáře v kódu piš anglicky
- Funkce a proměnné pojmenovávej anglicky v camelCase
- Soubory pojmenovávej v kebab-case
- Formátuj pomocí Prettier (2 mezery, středníky, jednoduché uvozovky)
- Při vytváření nových souborů přidej hlavičkový komentář s účelem souboru
